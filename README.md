# 課題解決型演習
## 実装内容について(鈴木担当分)
main.pyをベースとして、QAに解答する機能を実装しました。  
引数としては、
- qa_type: QAの種類を指定する引数: MultiChoice, YesNo(実装途中)
- qa_num: QAの種類を指定する引数: 3, 4

## パスの指定について
main.py, openai_gpt.pyにはデータセットへのパスを指定するための変数があります。  
openai_gpt.pyにはAPI_KEYを指定する必要があります。
